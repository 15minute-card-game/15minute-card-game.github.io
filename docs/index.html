ドロカルについて

もしも学校や家から徒歩15分くらいの超・ローカルなエリアのニュースを載せているフリーペーパーがあったら......。

ドロカルは、そんなどローカルなフリーペーパーのアリエナイおもしろ記事をプレイヤーたちで考えてカードから選んで遊ぶ大喜利型カードゲームです。

ドロカルの特徴

大喜利ゲーム

キーワードカードと見出しカードの組み合わせでアリエナイおもしろローカルニュースを考える大喜利ゲーム。

狭さを追求

徒歩15分圏内のコニュニティに特化したカードゲーム。狭い範囲のコミュティでするほど盛り上がる。
